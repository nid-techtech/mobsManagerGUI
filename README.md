# mobsManagerGUI
techTeCraft3に導入しているMobsManager PluginのconfigファイルをGUIで編集するアプリ  

このアプリはGoogle Antigravityを利用し、Gemini3による支援を受けて大渕凜が作成しました。  
このアプリはMobsManagerの非公式クライアントであり、Stellionix(MobsManager開発者)との関係はありません。
またMobsManager本体の商標権はStellionixに帰属します。

[MobsManager](https://github.com/Stellionix/MobsManager)
